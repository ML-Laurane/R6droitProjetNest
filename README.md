# R6droitProjetNest
 projet nest pour droit semestre 6

# Lancer le projet
se placer dans 'projet' puis:
```bash
npm run start
```
ou
```bash
npm run start:dev
```
-> le projet se lance sur le port 3001

--

se placer dans 'token' puis: 
```bash
npm run dev
```
-> le projet se lance sur le port 4500
